About

Drawing Practice is a personal project designed to help artists improve their skills through structured practice.

The application allows users to organize reference images, manage practice sessions, and use public Pinterest boards as image sources.

This website is provided for Pinterest API verification and application information.

Features

  • Drawing practice management
  • Reference image collections
  • Pinterest public board integration
  • Practice history management
